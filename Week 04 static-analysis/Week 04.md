# Week 04 - Static Analysis

## 📚 Topik
- Static Analysis
- Disassembler
- Decompiler
- Analisis Aliran Program
- Virtual Machine

## 📝 Ringkasan Pembelajaran

Pada pertemuan ini saya mempelajari teknik *Static Analysis* yang digunakan untuk menganalisis program tanpa menjalankannya secara langsung. Saya mengenal berbagai tools seperti disassembler, decompiler, dan hex editor yang membantu memahami struktur internal executable serta alur kerja suatu program. Selain itu, saya juga mempelajari pentingnya penggunaan Virtual Machine (VM) sebagai lingkungan yang aman dalam melakukan analisis terhadap file yang belum diketahui perilakunya, terutama ketika berkaitan dengan malware atau perangkat lunak yang berpotensi berbahaya.

## ✅ Apa yang Dipahami

Saya memahami bahwa Static Analysis memungkinkan analis memperoleh banyak informasi mengenai suatu program tanpa perlu menjalankannya, seperti struktur file, fungsi, library, dan alur program. Saya juga memahami fungsi dasar disassembler dan decompiler dalam membantu proses Reverse Engineering serta alasan penggunaan Virtual Machine untuk menjaga keamanan lingkungan analisis.

## ❓ Apa yang Masih Membingungkan

Saya masih ingin memahami bagaimana decompiler dapat menghasilkan kode yang menyerupai bahasa tingkat tinggi serta bagaimana optimasi compiler memengaruhi hasil proses dekompilasi. Saya juga ingin mempelajari lebih jauh mengenai teknik analisis terhadap executable yang telah diproteksi atau dipacking.
